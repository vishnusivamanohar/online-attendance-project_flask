About This Website

Nowadays, everything is moving online, so why shouldn't attendance be online too? Let's make it happen! This website is designed for teachers to record student attendance online instead of using traditional methods. To access the website, you must enter a password. If the password is correct, you will be redirected to the homepage.

The homepage includes the website's logo and key information about the online attendance system.
This website mainly consists of five pages:
1. Home Page
2. Students Page
3. Attendance Page
4. Mail Page
5. About / Learn More Page
Let's explore each of these pages in detail.

1. Home Page

The home page contains a welcome message and an introduction to the website.
At the top, you will find a navigation bar, which allows you to move to different pages: Students Page, Attendance Page, Mail Page, and About/Learn More Page.

2. Students Page

The Students Page follows the same theme as the Home Page, with a navigation bar for easy access to other sections. It provides key information and a "Learn More" button that redirects to the About page.

The Students Page allows you to perform three actions:
1. Add students
2. View students
3. Delete students
By clicking these buttons, you will be taken to a selection page where you can choose the branch, section, year, and number of students for the action.

If you choose "Add Students" or "Delete Students," after selecting the class and number of students, you will be provided with student information fields. After filling in the details, clicking "Submit Data" will insert all data into the database. You can verify student additions by clicking "View Students" on the main Students Page.

4. Mail Page

The Mail Page allows teachers to send notifications/messages through email to students. It involve two functions,
1. Send to all
2. Send to selected
First enter the massage/mail you want to send in the text area provided. After that,
By the option "Send to All" you can send your massage to all students in the collage without any selection.
By the option "Send to selected" you can send your massage to students in the selected class.

Working of the wedsite:

This website includes both frontend and backend development.
The entire frontend is built using HTML and CSS, providing a structured layout and styling for a user-friendly experience.
For the backend, I used Flask, a Python-based web framework that handles the logic and communication between the user interface and the database.
The database is created using MySQL, which stores all the necessary data for the website.
How It Works:
When a user interacts with the website, the provided data is stored in the MySQL database.
Flask acts as a bridge between the database and the frontend, retrieving and displaying the necessary information dynamically.
The Flask application runs in the background as a server, handling requests, processing data, and ensuring smooth communication between different components.
This combination of HTML, CSS, Flask, and MySQL ensures an efficient, responsive, and interactive online attendance system.
