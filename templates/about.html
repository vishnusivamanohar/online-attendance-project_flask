About This Website

Nowadays, everything is moving online, so why shouldn't attendance be online too? Let's make it happen! This website is designed for teachers to record student attendance online instead of using traditional methods. To access the website, you must enter a password. If the password is correct, you will be redirected to the homepage.

The homepage includes the website's logo and key information about the online attendance system.
This website mainly consists of five pages:
1. Home Page
2. Students Page
3. Attendance Page
4. Mail Page
5. About / Learn More Page

Working of the wedsite:

This website includes both frontend and backend development.
The entire frontend is built using HTML and CSS, providing a structured layout and styling for a user-friendly experience.
For the backend, I used Flask, a Python-based web framework that handles the logic and communication between the user interface and the database.
The database is created using MySQL, which stores all the necessary data for the website.
How It Works:
When a user interacts with the website, the provided data is stored in the MySQL database.
Flask acts as a bridge between the database and the frontend, retrieving and displaying the necessary information dynamically.
The Flask application runs in the background as a server, handling requests, processing data, and ensuring smooth communication between different components.
This combination of HTML, CSS, Flask, and MySQL ensures an efficient, responsive, and interactive online attendance system.
